How to use Entity Mappings

Entity mappings allow you to link the same logical entity across different systems. For example:

  • ACME Corporation = Jira project "ACME" = Salesforce Account "001ABC" = GitHub org "acme-corp"
  • John Doe = Jira user "jdoe" = Salesforce Contact "003XYZ"

AI agents use resolve_entity to automatically translate identifiers between systems when querying data.
